Dashboard WebGIS untuk membaca zona rawan kriminalitas di Yogyakarta berdasarkan titik kejadian klitih, CCTV, kepadatan penduduk, blind spot pemantauan, dan sumber berita pendukung.

Mulai dari Peta Lihat Analisis Buka Berita

Hasil pembobotan zona sudah terhubung ke halaman peta dan analisis. Gunakan dashboard ini sebagai media eksplorasi awal untuk melihat prioritas wilayah, bukan sebagai putusan tunggal tanpa validasi lapangan.

Struktur Dashboard

Alur dashboard dibuat runtut dari pemetaan, pembobotan, sampai interpretasi.

Pemisahan halaman membantu pengguna membaca konteks kriminalitas secara bertahap: melihat sebaran titik, memahami skor zona, membandingkan prioritas kecamatan, lalu meninjau sumber berita pendukung.

🏠

Halaman Utama

Berisi ringkasan tujuan, ruang lingkup data, akses cepat ke peta, analisis, dan berita pendukung kriminalitas.

Halaman Berita

Buka Berita
Komponen Analisis

Panel yang disiapkan untuk kebutuhan WebGIS kriminalitas.

Komponen dashboard disusun untuk membantu pembacaan risiko kriminalitas: informatif, mudah dipindai, memiliki filter, dan tetap menjaga konteks bahwa skor perlu diverifikasi dengan data lapangan.

🎚️

Parameter peta

Filter kecamatan, kategori risiko, radius buffer, cakupan CCTV, layer kejadian, network jalan, dan pilihan basemap.

🧮

Output analisis

Skor risiko, peringkat kecamatan, jumlah titik kejadian, cakupan CCTV, blind spot, tren waktu, dan rekomendasi awal.

🧾

Berita pendukung

Kartu berita dipakai untuk menautkan titik kejadian dengan sumber asli sehingga analisis spasial tetap memiliki konteks peristiwa.

⚙️

Alur kerja pengguna

1

Masuk Web

Pengguna membaca tujuan dashboard dan masuk ke halaman peta risiko.

2

Atur Parameter

Pengguna mengatur filter CCTV, kejadian, kecamatan, buffer, dan layer network jalan.

3

Proses Analisis

Data kejadian dan pembobotan zona diproses menjadi skor prioritas wilayah.

4

Baca Hasil

Diagram, tabel prioritas, insight spasial, dan narasi ditampilkan untuk dibaca bersama.

5

Cek Berita

Pengguna meninjau berita dan tautan sumber untuk mengecek konteks tiap kejadian.
