Struktur Halaman

Alur web dibuat jelas, bukan semuanya ditumpuk di satu halaman.

Pemisahan halaman membuat user lebih mudah memahami proses: masuk dari halaman utama, memilih parameter di peta, melihat hasil di halaman analisis, lalu membaca berita pendukung di halaman berita.

🏠

Halaman Utama

Berisi identitas web, tujuan, ringkasan fitur, struktur data, dan tombol menuju peta, analisis, serta berita.


Buka Utama

Buka Berita
Komponen Analisis

Panel yang disiapkan untuk kebutuhan WebGIS kriminalitas.

Komponen ini disusun supaya cocok untuk konteks semi-pemerintah: informatif, ringkas, punya filter, tetapi tetap tampil modern dan tidak kaku.

🎚️

Parameter peta

Filter kecamatan, jenis kriminalitas, kategori risiko, ambang skor minimum, opacity overlay, dan pilihan basemap.

🧮

Output analisis

Skor risiko, peringkat wilayah, total kasus demo, komposisi jenis kasus, dan narasi rekomendasi awal.

🧾

Berita pendukung

Kartu berita bisa diisi dari banyak sumber, lalu diarahkan ke link artikel asli untuk memperkuat konteks spasial.

⚙️

Alur kerja pengguna

1

Masuk Web

User membaca konteks dan memilih halaman peta.

2

Atur Parameter

User memilih jenis kasus, risiko, dan batas skor pada peta.

3

Proses Analisis

Parameter disimpan dan dikirim ke halaman analisis.

4

Baca Hasil

Diagram, tabel, dan narasi ditampilkan sesuai filter.

5

Cek Berita

User membuka berita pendukung dari berbagai sumber.
