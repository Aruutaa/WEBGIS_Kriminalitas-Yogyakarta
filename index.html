Upload GeoJSON lokal

Untuk sementara peta memakai data demo berupa titik/zona. Kalau kamu sudah punya batas kecamatan atau titik kejadian dalam format GeoJSON, upload dari panel ini. Layer akan langsung muncul di peta sebagai overlay tambahan.

Format: GeoJSON FeatureCollection. Untuk shapefile, ubah dulu ke GeoJSON melalui QGIS agar bisa dibaca langsung oleh browser.

Field yang ideal: kecamatan, kasus, risiko, kategori, kepadatan, kemiskinan, tpt.

Peta Interaktif

Panel WebGIS dengan filter, legenda, popup, dan ringkasan spasial.

Peta dibuat menggunakan Leaflet dengan basemap Esri, OSM, dan CartoDB. Basemap Google tidak dipasang langsung karena butuh API key dan ketentuan penggunaan yang berbeda.

Diagram, tabel, dan narasi siap diganti dengan data resmi.

Deskripsi analisis

Berdasarkan data demo, wilayah dengan skor risiko tertinggi ditampilkan sebagai zona prioritas awal. Interpretasi akhir tidak boleh ditarik dari data contoh ini, tetapi struktur visualnya sudah disiapkan untuk membaca keterkaitan antara jumlah kejadian kriminalitas, kepadatan penduduk, dan indikator sosial-ekonomi. Setelah data resmi dimasukkan, bagian ini dapat digunakan untuk menjelaskan kecamatan mana yang membutuhkan peningkatan patroli, pengawasan titik rawan, edukasi masyarakat, atau intervensi berbasis fasilitas publik.

Metodologi

Alur kerja analisis WebGIS zona kriminalitas.

Alur ini bisa dipakai sebagai bagian metode penelitian, baik untuk laporan praktikum, proyek WebGIS, maupun dashboard awal untuk instansi.

⚙️

Skema pengolahan data

1

Inventarisasi Data

Kumpulkan data kejadian kriminalitas, batas kecamatan, kepadatan penduduk, kemiskinan, pengangguran, serta fasilitas pendukung.

2

Standarisasi

Samakan nama wilayah, sistem koordinat, format tanggal, jenis kasus, dan struktur kolom agar bisa digabungkan.

3

Agregasi Spasial

Hitung jumlah kasus per kecamatan atau lakukan spatial join dari titik kejadian ke batas administrasi.

4

Normalisasi Indeks

Ubah setiap indikator menjadi skor sebanding, lalu gabungkan dengan bobot analisis kriminalitas atau MCDA.

5

Publikasi WebGIS

Tampilkan hasil ke peta interaktif, lengkapi filter, legenda, popup, chart, tabel, dan narasi rekomendasi.
