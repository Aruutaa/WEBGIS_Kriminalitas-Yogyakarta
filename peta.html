Auto-load dari file HTML

Layer dibaca otomatis dari file assets/data/kota_yogyakarta_admin.zip. ZIP asli sudah disederhanakan menjadi satu layer administrasi desa/kelurahan agar stabil dibaca browser. Jika ZIP gagal karena library, dashboard otomatis memakai GeoJSON cadangan.

Auto-load dari CSV

Data kejadian dibaca otomatis dari assets/data/Data Kejahatan Klitih.csv. Kolom yang dipakai: tanggal, bujur, lintang, kecamatan, dan URL sumber.

Catatan: koordinat CCTV masih memakai estimasi coordinateOverrides. Titik kejadian memakai koordinat dari CSV dan dinormalisasi otomatis jika desimal memakai koma atau lintang tertulis positif. Analisis jarak bersifat indikatif, bukan data resmi kepolisian.
